Summary


The purpose of this project is to see if we can identiy any weather paterns as city latitude gets closer to the equator. 500+ cities were randomly selected using Python's library citypi.


Once the cities were generated, we used Matplotlib to visualize: temperature, cloudiness, humitidy, and wind speed readings. This site provides the data and visualizations created as part of the analysis.
